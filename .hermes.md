## Release procedure (versionCode 2 / v1.0.x)
- Signing: `local.properties` (KEYSTORE_PATH / KEYSTORE_PASSWORD / KEY_ALIAS /
  KEY_PASSWORD) ‚Üí `release.jks`. Never commit either.
- Build: `./gradlew :app:assembleRelease` (R8 shrink + lintVital).
- Verify: `$ANDROID_HOME/build-tools/34.0.0/apksigner verify --verbose app/build/outputs/apk/release/app-release.apk`
  ‚Üí expect `Verified using v2 scheme: true`.
- Co-install caveat: debug and release SHARE the authenticator authority
  `com.unifiedcomms.authenticator`, so they CANNOT co-install on one device. To UI-verify
  the release: uninstall debug + test APKs ‚Üí install release ‚Üí launch/screencap ‚Üí reinstall
  debug + test APKs. Leave the emulator in its working (debug) state.
- Push commits + tag when green + fix verified (Keith workflow: no need to ask).
